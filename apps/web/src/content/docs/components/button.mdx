## Usage

```astro nocollapse
---
import { Button } from '@bejamas/ui/components/button';
---

<Button>Click me</Button>
```

## Link

You can use the `as` prop to render the component as an anchor.

```astro nocollapse
---
import { Button } from '@bejamas/ui/components/button';
---

<Button as="a" href="https://bejamas.com">Click me</Button>
```
