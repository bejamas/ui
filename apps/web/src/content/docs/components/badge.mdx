## Props

| Prop | Type | Default |
|---|---|---|
| <code>as</code> | `HTMLTag` | `"span"` |
| <code>variant</code> | `"default" \| "secondary" \| "destructive" \| "outline"` | `"default"` |
| <code>class</code> | `string` | `""` |
| <code>shape</code> | `"default" \| "pill"` | `"default"` |
| <code>size</code> | `"default" \| "sm"` | `"default"` |
